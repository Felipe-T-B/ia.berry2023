Insanely Fast Whisper

Descrição:

A ferramenta Insanely Fast Whisper é uma ferramenta de transcrição que utiliza a tecnologia Whisper Large V3 da OpenAI para transcrever rapidamente arquivos de áudio. Inclui um script CLI e uma API de inferência para ajudar a automatizar o processo. Ele também possui diversas otimizações, como lote, tamanho do feixe e atenção de flash para ajudar a acelerar o processo. Além disso, possui um roteiro e uma vitrine da comunidade para ajudar os usuários a aproveitar ao máximo a ferramenta. Nota: Este é um repositório GitHub, o que significa que é um código que alguém criou e disponibilizou publicamente para qualquer pessoa usar. Essas ferramentas podem exigir algum conhecimento de codificação.
